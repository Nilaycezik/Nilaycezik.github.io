Basic Java

What is JVM,JRE,JDK?

JVM: Java Virtual Machine

It is an abstract intermediary layer between the Java application and the platform on which it executes. JVM functions differently depending on the platform. That is, it depends on the platform (Windows, Linux, or Mac) on which it was created. There are a variety of implementations to choose from. The JVM is unable to recognize files with the.java suffix. in place of Can read and write files ending in.class. Because bytecode is present in these files. This function is possible because of this feature. In Java, the "write once, run everywhere" functionality may be employed.

JRE: Java Runtime Enviroment

With components that make it possible for Java-based apps and applets to run. It gives the JRE libraries to use. Bytecodes that have been compiled do not execute directly on the CPU. They must be interpreted by the CPU. Bytecodes are translated into readable machine code by the JRE. Java bytecodes, in reality, work on all systems owing to the JRE. After the Java codes are compiled, the JRE produces the Java bytecode field, which is called an intermediate language. For all operating systems, these bytecodes are the same.

JDK: Java Runtime Enviroment

It is a must-download component for every developer in Java. It acts as both an interpreter and a compiler. In summary;

JRE = JVM + Java Libraries

JDK = JRE + Compiler + Debugger

Java Type Casting

Widening Casting

Automatic casting: int to double

Example

public class Main {
    public static void main(String[] args) {
        System.out.println("Java Type Casting");
        System.out.println("Widening Casting");

        int numberInt = 8;
        double doubleNumber = numberInt; // Automatic casting: int to double

        System.out.println(numberInt);      // Outputs 8
        System.out.println(doubleNumber);   // Outputs 8.0
    }
}

Narrowing Casting

Manual casting: double to int

Example

public class Main {
    public static void main(String[] args) {

        //Narrowing Casting
        double doubleNewNumber = 10.95d; // Manual casting: double to int
        int numberNewInt = (int) doubleNewNumber;

        System.out.println("Narrowing Casting");
        System.out.println(numberNewInt);      // Outputs 10
        System.out.println(doubleNewNumber);   // Outputs 10.95
    }
}

Java Switch-Case
Example

    public static void main(String[] args) {

    //Java Switch-Case
    System.out.println("Java Switch-Case");
    int day = 5;
    switch (day) {
        case 1:
            System.out.println("Monday");
            break;
        case 2:
            System.out.println("Tuesday");
            break;
        case 3:
            System.out.println("Wednesday");
            break;
        case 4:
            System.out.println("Thursday");
            break;
        case 5:
            System.out.println("Friday");
            break;
        case 6:
            System.out.println("Saturday");
            break;
        case 7:
            System.out.println("Sunday");
            break;
    }
    // Outputs "Friday" (day 5)
    }
}
